Job Summary

KLA is hiring Software QA Engineers to build and enhance automated testing solutions for next-generation semiconductor software products. You will develop automation frameworks, perform functional, regression and API testing, collaborate with global engineering teams, and help ensure world-class software quality using modern testing practices and AI-assisted automation.

Educational Qualification

  • B.E. / B.Tech / M.E. / M.Tech / M.Sc / MCA.
  • Computer Science or related Engineering discipline.
  • Eligible Batch: 2024 / 2025 / 2026.
  • 0–1 years of experience. Freshers can apply.

Key Responsibilities

  • Design and implement software test automation frameworks.
  • Develop functional, regression, performance and reliability test suites.
  • Automate UI and API testing for enterprise software products.
  • Implement AI-assisted testing including intelligent test generation and self-healing automation.
  • Evaluate and improve automation coverage across products.
  • Collaborate with software, algorithms, application and manufacturing teams.
  • Participate in software deployment and customer issue troubleshooting.
  • Document testing strategies, best practices and automation reports.
